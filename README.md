# AgriSphere

AgriSphere is a digital farming assistant platform that gives small farmers instant, affordable, and accessible agronomic intelligence using AI, IoT, and mobile technology.

## Key Features

### AI Vision

Detects crop diseases using a smartphone camera — early detection prevents crop loss.

### Smart Sensors (IoT)

Monitors soil moisture, temperature, humidity, and pH in real time — no more guesswork in watering or fertilizing.

### Predictive Intelligence

Uses AI to recommend actions and predict yields — farmers can plan ahead and avoid losses.

### Location-Based Intelligence

Uses weather and regional data to provide advice specific to the farmer's environment.

## Tech Stack

- **Mobile App**: React Native (Expo)
- **AI Backend**: Python (Flask) + Google Gemini AI
- **IoT**: ESP32/ESP8266 sensors
- **Database**: Firebase (Firestore, Auth)

## Getting Started

```bash
# Install dependencies
npm install

# Start the app
npx expo start

# Start the backend
cd models
pip install -r requirements.txt
python app.py
```
